# Blog team ranking app

naver cafe, blog의 랭크를 저장하는 프로그램이다.
blog team member들의 블로그 포스팅 순위를 기록한다.

설정한 google spread sheet에서 keyword와 member를 가져와서 keyword로 검색시 첫번째
페이지에 들어가 있는 멤버를 기록한다.
